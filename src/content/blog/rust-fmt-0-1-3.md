> 注: v0.1.3 のリリース日は **2026年2月8日** です。

rust-fmt 0.1.3 は `rustfmt` で Rust コードを整形する VS Code 拡張の小さく集中したアップデートです。目的は拡張をデフォルトとして使える状態にし、手動設定の手間を減らすことで、グローバル／ワークスペースどちらでも rust-fmt をフォーマッタとしてすぐ設定できるクイックコマンドを追加し、Rust が rust-fmt をデフォルトフォーマッタとして使っていない場合はスマートなプロンプトが表示されます。

大規模プロジェクトでもワークスペース整形が速くなり、待ち時間を減らしつつミニマルなワークフローを保ちます。
